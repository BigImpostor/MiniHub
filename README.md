# WanAndroid
基于www.wanandroid.com提供的API实现的第三方客户端。
使用Rxjava+Retrofit进行网络请求。
把较为庞大的Activity使用MVP模式解耦。
目前实现的功能有，注册，登录，搜索，分类导航等功能。
